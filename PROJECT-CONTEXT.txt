THE MYSTERY MAIL — VAULT WEBSITE (PROJECT CONTEXT)
====================================================
Last updated for: ongoing development / AI & human handoff

WHAT THIS PROJECT IS
---------------------
This repository powers "The Mystery Mail Vault" — a small static player-facing site
for physical mystery-mail / envelope games sold via the main Wix store
(themysterymail.com). It is separate from the Wix storefront: code lives here,
deployed to GitHub Pages (and optionally a subdomain like vault.themysterymail.com).

The playable site files live in: themysterymail-vault/

Main flow (envelope username → multi-cell password → fake "Windows" directory with .txt):
  - themysterymail-vault/index.html
  - themysterymail-vault/password.html?game=<id>
  - themysterymail-vault/directory.html?game=<id>
  - Game definitions: themysterymail-vault/data/games.json

Extra standalone flow (Case 1, second beat — numeric only → "Decode letter" + image):
  - themysterymail-vault/case-1-vault-2.html
  - Not linked from index.html — intended access: QR on physical puzzle (or direct URL only).
  - Images per beat: themysterymail-vault/images/<case-slug>/ (see images/README.txt)

GitHub repo (source): https://github.com/a-ray22/themysterymail.vault
